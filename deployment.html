Deploy Your Website

Follow these simple steps to deploy your INFERNO MACH website to GitHub Pages:

Run the Deployment Script

Double-click the DEPLOY_TO_GITHUB.bat file in your website folder. This will:

  • Set up Git configuration
  • Initialize the repository
  • Add and commit all files
  • Open the GitHub new repository page

Create GitHub Repository

On the GitHub page that opens:

  1. Name your repository: Inferno_Mach
  2. Make it Public
  3. Do NOT initialize with a README
  4. Click Create repository

Connect and Push Code

Copy and run these commands in Git Bash or terminal:

cd "d:\KAYAM_SAI_KRISHNA\InfernoMach\website"
git remote add origin https://github.com/YOUR_USERNAME/Inferno_Mach.git
git branch -M main
git push -u origin main

Replace YOUR_USERNAME with your actual GitHub username.

Enable GitHub Pages

After pushing your code:

  1. Go to your repository Settings
  2. Navigate to Pages section
  3. Select "Deploy from a branch"
  4. Choose main branch and "/" folder
  5. Click Save

Your website will be live at: https://YOUR_USERNAME.github.io/Inferno_Mach/
