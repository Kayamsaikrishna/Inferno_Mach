AI Measurement Tool

Precision measurement using optical imaging and computer vision techniques

Our AI Measurement Tool is a comprehensive industrial-grade application for precision measurement of coating and layer thickness in manufacturing environments. Using optical imaging and computer vision techniques, this solution provides accurate measurements essential for quality control in various industries.

Key Features

  • Real-time Measurement: Capture and measure layer thickness using Sentech cameras
  • Multi-platform Support: Desktop application (PyQt5) and Web interface (FastAPI/React)
  • Database Integration: PostgreSQL backend for robust data storage and retrieval
  • IQC Management: Track quality control requests and materials
  • Excel Reporting: Generate comprehensive reports with VBA-compatible formatting
  • Calibration System: Precise measurement calibration for accuracy
  • Personnel Logging: Track user actions and activities for compliance
